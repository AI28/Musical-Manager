Modulele de inregistrare si autentificare

Modulele de inregistrare si autentificare sunt primele componente ale aplicatiei cu care poate interactiona utilizatorul. Acestea au o interfata grafica minimala dar eleganta, constand din cate un simplu:

  • CSS (Cascading Style Sheets) is a style sheet language used for describing the presentation of a document written in a markup language like HTML
    • Functionalitatea consta in trimiterea datelor din formulare la server folosind o cerere asincrona
  • Ajax is a set of web development techniques using many web technologies on the client side to create asynchronous web applications.

    Iar odata ajunse la server, acestea sunt procesate si clientului ii este intors un raspuns corespunzator. Tehnologii folosite pentru implementarea modulului:

      Dashboard

      In dashboard utilizatorul poate vizualiza ultimele activitati consemnate in sistem care pot reprezenta fie auditia muzicala a unui cantec de catre un utilizator sau grup de utilizatori, fie incarcarea in sistem a unei noi productii muzicale sau compilatii. Un utilizator autentificat poate reactiona la aceste activitati, reprezentate de content cards, lasand o apreciere sau un comentariu; un utilizator anonim poate doar sa vizualizeze dashboardul.

      Pagina de administrare a contului

      Un utilizator autentificat isi poate gestiona istoricul activitatii listele de melodii preferate si informatii despre cont dintr-o pagina web dedicata.
    Istoricul auditiilor muzicale

    Un utilizator autentificat isi poate vizualiza istoricul auditiilor poate sterge auditii din sistem si comentarii de la respectivele auditii. In plus, utilizatorul poate serializa in formatele CSV si PDF istoricul auditiilor muzicale.

    Tehnologii folosite:

  • TCPDF is a free Open Source PHP class for generating PDF documents.
    • Exportarea din sistem si importarea in sistem a playlist-urilor utilizatorului

    Un utilizator autentificat poate introduce in sistem prin incarcarea unui document in formatul , un playlist propriu; de asemenea, un utilizator autentificat poate serializa in format JSPF playlisturi existente in sistem.

    Tehnologii folosite:

  • JSPF stands for JSON Shareable Playlist Format and is a data-format for sharing music playlists.
    • Modificarea detaliilor contului

    Din aceasta sectiune utilizatorul isi poate schimba numele contului, parola, poza de profil si isi poate radia contul din sistem.
